BorrowIT Project

A gadget rental platform UI designed to connect people who own unused electronics with people who need them temporarily.

Project Overview

BorrowIT is a platform designed to allow users to rent electronic gadgets such as earbuds, laptops, and power banks. The goal of this project is to reduce unused resources and provide affordable access to technology.

Key Features

  • User Login & Signup UI
  • Product Rental System
  • Modern Chat Interface
  • Clean UI/UX Design
  • Responsive Layout
